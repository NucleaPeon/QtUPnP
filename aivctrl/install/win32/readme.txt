To create an Windows installer an InnoSetup script is provided.

1. Install InnoSetup e.g. http://www.jrsoftware.org/isinfo.php
2. Launch build.cmd to create the "build" folder containing all files to run AIVCtrl. At this stage double click on aivctrl.exe
   start the application.
3. Launch InnoSetup with the file aivctrl.iss. An installer is created in Ouput folder.
